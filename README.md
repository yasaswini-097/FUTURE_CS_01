# FUTURE_CS_01
##  Project Overview
This project demonstrates a Web Application Vulnerability Assessment conducted on a demo application for educational purposes.

The assessment was performed to identify security misconfigurations, session vulnerabilities, and other potential risks using industry-standard tools.

---

 ## Objectives
- Identify security vulnerabilities in a web application
- Classify risks based on severity levels
- Provide remediation recommendations
- Document findings in a professional report format

---

 ## Tools Used
- Nmap
- OWASP ZAP
- Browser Developer Tools
- Canva (Report Design)

---

##  Assessment Type
Passive Web Application Security Testing
